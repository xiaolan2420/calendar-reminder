# 日历提醒服务

## 项目介绍
实现了注册、登录、日程的增删改查和邮箱通知
登录和注册 使用了JWT鉴权  只有该用户才可以管理本人的日程
日程的提醒 简易实现了一次性日程、每天重复性日程和周一到周五提醒的日程
